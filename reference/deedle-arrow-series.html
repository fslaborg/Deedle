Series Module

Namespace: Deedle.Arrow
Assembly: Deedle.Arrow.dll

Arrow-specific functions on Deedle Series values. Open Deedle.Arrow and then call these as Series.toArrowArray, Series.ofArrowArray, etc.

Functions and values

Function or value Description

ofArrowArray arr

Full Usage: ofArrowArray arr

Parameters:
    arr : IArrowArray

Returns: Series<int, obj>

Convert an Apache Arrow IArrowArray to a Deedle Series<int,obj>. Row keys are 0-based integers. Arrow null values become Deedle missing values. The values are boxed as obj; use Series.map to obtain a typed series.

arr : IArrowArray
Returns: Series<int, obj>

toArrowArray series

Full Usage: toArrowArray series

Parameters:
    series : Series<'K, 'V>

Returns: IArrowArray
Type parameters: 'K, 'V (requires equality)

Convert a Deedle Series to an Apache Arrow IArrowArray. The series element type determines the Arrow array type (same mapping as Frame.toRecordBatch). Missing values are encoded as Arrow validity-bitmap nulls.

series : Series<'K, 'V>
Returns: IArrowArray
